# Next.js Minimal Starter

A minimal Next.js 15 starter template with TypeScript and Tailwind CSS.

## Getting Started

First, install dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Project Structure

- `app/` - Application routes and pages
  - `page.tsx` - Home page
  - `layout.tsx` - Root layout

## Technologies Used

- Next.js 15
- TypeScript
- Tailwind CSS
